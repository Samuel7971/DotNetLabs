# 📚 Decisões importantes

Durante esta Sprint foram tomadas as seguintes decisões:

- Documentação faz parte do produto.
- Arquitetura antes da implementação.
- Todo Lab possuirá documentação própria.
- A plataforma será composta por Blazor + API.
- A solução seguirá uma organização baseada em responsabilidades.
- Toda decisão arquitetural relevante deverá possuir uma ADR.

---

Ao final desta Sprint, o projeto possui:

- Identidade definida.
- Objetivos claros.
- Arquitetura documentada.
- Roadmap inicial.
